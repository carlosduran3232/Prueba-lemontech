## Instalation

```bash
npm i
```

## Usage

```
Hay dos opciones 

1 Correrlo desde el cypres:
    * .\node_modules\.bin\cypress open
    * y luego hacer click en el archivo prueba-lemon.js

2 Correrlo desde la consola
    * npm run cypress:run

La diferencia entre ellos es que en el 1 pueden ver paso a paso desde el navegador abierto y en la segunda podrian correr todos los sets de prueba sin necesidad de abrir un navegador
```
